# WHMCS Scrape Website Ownership

# Set Up
1. Clone this repository
2. Make sure you have Selenium and Python installed

## Usage
- First, update the config file with your WHMCS Username and password.
- Then run ____.py to scrape website data

Note: The data is organized as a csv file where the first column is the admin user email and the subsequent columns contain the domains under which they are listed as an admin.
